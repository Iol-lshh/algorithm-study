# 다익스트라 알고리즘
- edge 또는  node에 값이 있을 경우, 최단 경로를 구하는데 사용
- 가장 싼 가격 순서대로, BFS를 진행

1. 가장 싼 가중치의 정점을 찾는다.
2. 이웃 정점에 대해, 더 싼 경로가 있는지 확인한다.
    - 있다면 가격을 수정한다.
3. 최종 경로를 계산한다.

- 단, 음의 가중치는 처리하지 못한다.
    - 벨만 포드 알고리즘

- 가중치 weight
- 균일 그래프 unweighted graph
- 가중 그래프 weighted graph
- 방향성 비순환 그래프 directed acyclic graph


- 세 개의 테이블이 필요
1. 그래프
2. 가격
3. 부모


- 한 시작점에서 다른 노드들에 이르는 최소 값들을 연산하는 방식
